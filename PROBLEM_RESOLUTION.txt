## Configuration Personnalisée

Modifier `config.yaml` pour ajuster:

```yaml
dataset:
  max_events: null          # Tous les cascades (ou 100 pour test rapide)
  min_nodes: 2              # Cascades minimum

casdi:
  alpha: 0.35               # Poids spectral
  beta: 0.25                # Poids core density
  gamma: 0.25               # Poids community bridging
  delta: 0.15               # Poids centralization

evaluation:
  random_state: 42          # Seed reproductibilité
  train_test_split: 0.8
  n_folds: 5
  n_bootstrap: 1000
```

## Support & Troubleshooting

### Si le pipeline crash:
1. Vérifier espace disque: `Get-Volume`
2. Vérifier logs: `outputs/logs/run.log`
3. Relancer avec subset réduit: `dataset.max_events: 100`

### Si "No space left":
1. Supprimer old project outputs
2. Nettoyer cache: `Remove-Item -Recurse -Force __pycache__`
3. Redémarrer PowerShell
